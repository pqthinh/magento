### git command

1: Clone 

`git clone https://github.com/pqthinh/magento.test.git`

2: Check out to master branch

`git checkout master`

3: Pull

`git pull`

4: Commit code 

```git
git add .
git commit -m "message"
git push
```

5: Commit force

``` git
git add .
git commit --amend --no-edit
git push -f
```
